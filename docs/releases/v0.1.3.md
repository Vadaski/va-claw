# va-claw v0.1.3

## What is va-claw?

`va-claw` turns Claude Code or Codex into a persistent local agent by adding memory, identity, and a wake-loop daemon without introducing a new gateway or cloud dependency.

## Highlights

- **Memory CRUD + Ebbinghaus reinforcement**: store, list, update, forget, recall, consolidate, and reflect memories in local SQLite with forgetting-curve based strength updates.
- **Identity injection**: inject a persistent persona and system prompt into Claude Code or Codex startup instructions.
- **Wake-loop daemon**: run scheduled autonomous wake cycles in the background so the agent can resume work without manual prompting.
- **Channels**: connect the same local agent flow to Discord, Telegram, and Slack.
- **Skills**: install, list, search, and inject reusable markdown skills into wake runs and interactive sessions.
- **Global install bundle**: ship the CLI through the locked `bin/va-claw` -> `dist/va-claw-bundle.mjs` esbuild bundle path for stable global installs.

## Notes for this release

- Node 22 is the supported runtime.
- Everything runs offline-first with no cloud dependency added by this release.
- This tag gives users a stable GitHub release reference that matches the published package version `0.1.3`.
